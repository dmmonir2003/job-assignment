# Task - Animated Landing Page

A modern, animated landing page built with Next.js, featuring smooth animations and a responsive design.

## Technology Overview

This project utilizes the following technologies:

- **Next.js 16**: A React framework for production, providing server-side rendering, static site generation, and optimized performance.
- **React 19**: The latest version of React, offering improved performance and new features like concurrent rendering.
- **TypeScript**: Adds static typing to JavaScript, helping catch errors early and improving code maintainability.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development with a focus on responsive design.
- **Framer Motion**: A library for creating smooth animations and transitions in React applications.
- **Radix UI**: A set of unstyled, accessible UI components that can be styled with Tailwind CSS.
- **ESLint**: A tool for identifying and fixing linting issues in JavaScript/TypeScript code.

## Why These Frameworks and Technologies?

- **Next.js**: Chosen for its excellent performance optimizations, SEO benefits through server-side rendering, and ease of deployment on Vercel. It's ideal for landing pages that need fast loading times.
- **React**: Provides a component-based architecture, making the UI modular and reusable. React 19 brings performance improvements that enhance user experience.
- **TypeScript**: Ensures type safety, reducing runtime errors and making the codebase more robust, especially as the project grows.
- **Tailwind CSS**: Allows for rapid prototyping and consistent styling without writing custom CSS. Its utility classes make it easy for beginners to style components.
- **Framer Motion**: Enables complex animations with simple syntax, adding visual appeal to the landing page without compromising performance.
- **Radix UI**: Provides accessible, customizable components that integrate seamlessly with Tailwind, ensuring the app is usable by everyone.
- **ESLint**: Maintains code quality and consistency across the team.

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd task-animated-landing-page
   ```
2. Install the dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   # or
   bun install
   ```

### Development

- To start the development server:
  ```bash
  npm run dev
  # or
  yarn dev
  # or
  pnpm dev
  # or
  bun dev
  ```
- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
- You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

### Production

- To build the project for production:
  ```bash
  npm run build
  # or
  yarn build
  # or
  pnpm build
  # or
  bun build
  ```
- To start the production server:
  ```bash
  npm start
  # or
  yarn start
  # or
  pnpm start
  # or
  bun start
  ```
